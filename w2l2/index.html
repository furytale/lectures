CSS BOX MODEL

Calculation formula

Total width:
margin-right + border-right + padding-right + width + padding-left + border-left + margin-left
Total height:
margin-top + border-top + padding-top + height + padding-bottom + border-bottom + margin-bottom

Width

  • Default width depends on the element's display value.
  • Block elements have a default width of 100%.
  • Inline elements doen' accept width property, but can accept their corresponding values

Height

  • Default Depends on the element' content
  • Inline elements doen' accept height property, but can accept their corresponding values

Margin

  • Depentds on browser
  • Works normal for blocks
  • Vertical margins are not acceptable to inline elements

Padding

  • Depentds on browser
  • Works normal for blocks
  • Falls inside the element
  • Works for inline elements, on all sides

Borders

  • Requires width, style, and color
  • Can be defined using common CSS units of length
  • Can be placed on one side of an element at a time
  • border-radius property
  • They move in a clockwise fashion starting at the top left of an element

Box Sizing

  • Calculation
  • Accepts: content-box, padding-box and border-box

Content-box

  • content-box value is the default value
  • The size of an element begins with the width and height properties, and then any padding, border, or margin property values are added
div {

-webkit-box-sizing: content-box;
-moz-box-sizing: content-box;
box-sizing: content-box;

}

Padding Box

  • Alters the box model by including any padding property values within the width and height of an element
  • if an element has a width of 400 pixels and a padding of 20 pixels around every side, the actual width will remain 400 pixels
div {

-webkit-box-sizing: padding-box;

}

Border Box

  • Alters the box model so that any border or padding property values are included within the width and height of an element
  • if an element has a width of 400 pixels, a padding of 20 pixels around every side, and a border of 10 pixels around every side, the actual width will remain 400 pixels
div {

-webkit-box-sizing: border-box;

}

Box sizing

  • Border box is the best solution
  • Doesn' have support in all browsers
